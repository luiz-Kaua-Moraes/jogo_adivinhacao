Jogo de Adivinhação em C

Este é um mini projeto desenvolvido em **linguagem C** para praticar lógica de programação.

O programa gera um número aleatório entre **1 e 100**, e o jogador precisa adivinhar qual é o número escolhido pelo computador.

O jogo informa se o número digitado é **maior ou menor** que o número correto e também mostra o **número de tentativas** necessárias para acertar.

---

##  Funcionalidades

- Geração de número aleatório
- Sistema de dicas (maior ou menor)
- Contador de tentativas
- Validação de entrada do usuário

---

##  Conceitos praticados

- Estrutura de repetição `while`
- Estruturas condicionais `if` e `else`
- Geração de números aleatórios com `rand()`
- Contadores e lógica de programação
- Entrada e saída de dados (`scanf` e `printf`)
